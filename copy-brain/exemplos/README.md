**Critério para salvar:** toda análise com veredicto OURO ou PRATA, OU análise BRONZE/REPROVADO com reescritas completas que demonstrem transformação real.

## Nomenclatura

```
[YYYY-MM-DD]-[nicho]-[tipo]-[veredicto].md

Exemplos:
2026-05-22-implantodontia-lp-ouro.md
2026-05-22-implantodontia-ad-reprovado-reescrita.md
2026-05-30-agencia-email-prata.md
```
