# Workflow Completo de Auditoria Neural de Copy

**Brain Copy OS — Protocolo Oficial**
**Versão:** 1.0 | Tempo estimado: 45-90 min por copy completa

---

## OVERVIEW DO PROCESSO

```
FASE 0: CONTEXTO (10 min)
   ↓ ICP + Nível de consciência + Tipo de copy
FASE 1: LEITURA DIAGNÓSTICA (5 min)
   ↓ Impressão inicial + pontos de abandono
FASE 2: 7 FILTROS CEREBRAIS (30-60 min)
   ↓ F1 → F2 → F3 → F4 → F5 → F6 → F7
FASE 3: SCORING + DIAGNÓSTICO (10 min)
   ↓ Score por filtro + Score geral + Top 3 erros críticos
FASE 4: REESCRITA CIRÚRGICA (15-30 min)
   ↓ Apenas os pontos que falham nos filtros
OUTPUT: Relatório de Auditoria Neural
```

## FASE 1 — LEITURA DIAGNÓSTICA

**Leia a copy uma vez, inteiramente, sem parar.**

Enquanto lê, marque com:
- `[PARA]` = Onde você pararia de ler se fosse o ICP
- `[CONFUSO]` = Onde você não entendeu imediatamente
- `[FRACO]` = Afirmação que não acreditaria sem prova
- `[FORTE]` = Seção que capturou atenção genuinamente
- `[FALTOU]` = Algo que você esperava que não apareceu

**Esses marcadores orientam onde aplicar mais atenção nos filtros.**

4. Pule a reescrita por agora (fase 4)

## FASE 4 — REESCRITA CIRÚRGICA

**Regra:** reescrever APENAS as seções que falham em filtros CRÍTICOS para o tipo de copy. Não reescrever o que já funciona.

**Processo para cada erro crítico:**

```
1. Identificar o erro exato (código + linha)
2. Ler o padrão correto no filtro correspondente
3. Escrever 2-3 versões alternativas
4. Escolher a versão mais fiel ao voice do ICP
5. Inserir e verificar se o filtro passa
```

**Regra de parada:** quando todos os filtros CRÍTICOS passam no threshold, a copy está pronta.
