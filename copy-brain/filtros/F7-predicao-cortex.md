# F7 — FILTRO DE PROCESSAMENTO PREDITIVO

**Região cerebral:** Hierarquia cortical completa (top-down) + Córtex Pré-Frontal + Córtex Temporal Superior + Cerebelo (predição motora)
**Janela crítica:** Ao longo de toda a copy — é o filtro de arquitetura narrativa
**Mecanismo:** Predictive Processing (Friston, 2010) + Active Inference + Prediction Error Minimization

---

## O QUE ESTE FILTRO AVALIA

Karl Friston (UCL) demonstrou que o cérebro é fundamentalmente uma máquina de predição. Em cada momento:
1. Gera uma predição do que vai perceber a seguir
2. Recebe o input real
3. Compara predição vs. realidade
4. Usa o ERRO DE PREDIÇÃO para atualizar o modelo interno

**Implicação radical para copy:**

> Copy que o ICP consegue prever = processada automaticamente, sem atualização de crenças = esquecida
> Copy que viola produtivamente a predição = gera erro de predição = atualização de crenças = memorável + persuasiva

O F7 não é um "filtro de estilo". É o filtro de **ARQUITETURA DA MUDANÇA DE CRENÇA**.

## OS 4 PADRÕES DE PREDIÇÃO EM COPY

### P1 — CONFIRMAÇÃO DE EXPECTATIVA (Processamento Fluente — baixa persuasão)
O ICP leu antes algo similar. A copy confirma o que ele esperava.

**Efeito:** Processamento eficiente, sem atualização de crença.

**Quando usar:** Partes técnicas, explicações de processo, garantias (onde fluência = credibilidade).

**Exemplo:**
> "Trabalhamos com anúncios no Meta, Google e WhatsApp para gerar leads qualificados para sua clínica."

→ ICP esperava: "Agência de marketing digital faz anúncios." Confirmado. Zero surpresa. Zero mudança de crença.

### P2 — VIOLAÇÃO PRODUTIVA (Prediction Error + Resolução = Máxima Persuasão)
A copy viola uma predição do ICP de forma que a resolução o leva para um novo entendimento.

**Estrutura:**
1. Ativar a predição (o que o ICP acredita ser verdade)
2. Violá-la com evidência específica
3. Oferecer resolução que atualiza a crença
4. A nova crença favorece a ação que você quer

**Exemplo:**
> **Predição ativada:** "Agências prometem leads e entregam formulários de contato vagos"
> **Violação:** "Nossos clientes não recebem leads. Recebem consultas de avaliação agendadas — com nome, WhatsApp confirmado e interesse específico em implante."
> **Resolução:** "A diferença é que nós trabalhamos com qualificação pré-agendamento — o paciente só entra na agenda se passar em 3 critérios de qualificação financeira."
> **Nova crença:** "Isso pode ser diferente do que já tentei antes."

### P3 — VIOLAÇÃO IMPRODUTIVA (Confusão — abandono)
A copy viola a predição mas não oferece resolução satisfatória.

**Efeito:** Confusão → frustração → abandono.

**Como identificar:** Se o ICP teria a reação "Mas então o que você faz afinal?", é P3.

### P4 — GAP PREDITIVO (Curiosity Loop — alta motivação para continuar lendo)
A copy cria uma predição parcial — suficiente para criar expectativa, mas sem revelar a resolução.

**Efeito:** Sistema dopaminérgico ativa para "completar o gap". O ICP continua lendo.

**Técnica:**
> "Existe um motivo pelo qual a maioria das clínicas que tentam fazer anúncios por conta própria gastar mais dinheiro quanto mais tempo passa — e não é o que você pensa."

→ ICP prediz resposta parcialmente → gap criado → precisa continuar para resolver.

## A ESTRUTURA PREDITIVA DE COPY DE ALTA CONVERSÃO

O mapa preditivo ideal de uma copy ou VSL:

```
ENTRADA: Predição inicial do ICP sobre o conteúdo (geralmente: "mais uma agência prometendo leads")

VIOLAÇÃO 1 (Hook): Viola a predição mais óbvia → abre gap preditivo #1
         ↓
AGITAÇÃO: Confirma a dor (predição correta do ICP sobre o problema → valida que você entende)
         ↓
REVELAÇÃO: Viola predição sobre a CAUSA do problema (não é o que você pensa)
         ↓
MECANISMO: Nova predição (como funcionaria) + gap preditivo #2 ("mas como especificamente?")
         ↓
PROVA: Fecha gap #1 e #2 com casos reais → nova crença formada
         ↓
OFERTA: Viola predição sobre preço (mais acessível ou mais específico que esperado)
         ↓
GARANTIA: Viola predição sobre risco ("se não funcionar, devolvemos tudo" vs. expectativa de risco)
         ↓
CTA: Confirma nova crença → ação parece inevitável

SAÍDA: Crença atualizada + motivação para agir
```

## O TESTE DO "E DAÍ?"

Para cada seção da copy, o ICP instintivamente processa: "E daí para mim?"

Essa é a predição ativa de relevância pessoal. Se a copy não responde, o ICP abandona.

**Como aplicar:**
1. Escreva sua copy normalmente
2. Depois de cada parágrafo, escreva "[ICP]: E daí?"
3. Se a próxima frase não responde explicitamente → refatorar

---

## O TESTE DA "OBJEÇÃO ANTECIPADA"

O DLPFC processa objeções como predições de resultado negativo. Copy que antecipa objeções "desativa" predições negativas.

**Processo:**
1. Liste as 5-7 predições negativas do ICP sobre sua oferta
2. Identifique em qual seção da copy cada predição vai surgir
3. Certifique-se de que a copy resolve ANTES de o ICP formular a objeção

**Predições negativas universais por seção:**
| Seção | Predição negativa do ICP |
|-------|--------------------------|
| Hook | "Mais uma promessa exagerada" |
| Mecanismo | "Isso parece complicado/novo demais" |
| Prova social | "Pode ser falso" / "Minha situação é diferente" |
| Preço | "Caro" / "E se não funcionar?" |
| CTA | "Não tenho certeza se estou pronto" |

## CTA COMO CONCLUSÃO PREDITIVA (Inevitable CTA)

O CTA de máxima conversão é aquele que o ICP sente que ELE MESMO concluiu, não que foi "vendido".

**Estrutura de CTA Inevitável:**
1. Summarize as crenças atualizadas: "Se você chegou até aqui, você já sabe que..."
2. Projete a consequência lógica: "...a única questão real é se você vai implementar isso sozinho ou com ajuda especializada"
3. Apresente a ação como conclusão: "A próxima etapa natural é agendar uma conversa de 30 minutos para ver se faz sentido para sua clínica específica"

**Anti-padrão (CTA como imposição):**
> "Clique agora e garanta sua vaga antes que acabe!!!"

→ O ICP não foi levado a uma conclusão preditiva → o CTA parece externo, não interno → resistência

## ERROS MAIS COMUNS NESTE FILTRO

### Erro P1 — "Predictable Hook"
Hook confirma exatamente o que o ICP esperava de uma agência/produto do segmento.
- Fix: Identifique a predição mais óbvia do ICP → viole-a nos primeiros 15 palavras

### Erro P2 — "Violation Without Resolution"
A copy diz algo surpreendente mas não explica por quê ou como → P3 (confusão)
- Fix: Toda violação deve ter resolução dentro do mesmo bloco de copy

### Erro P3 — "Generic Cause Frame"
A causa do problema é "você não tem sistema" ou "não está fazendo marketing direito" → ICP já sabia isso
- Fix: Revelar causa não-óbvia e específica: "O problema não é você não ter leads — é que os leads que você tem não passam por qualificação financeira antes de entrar na agenda"

### Erro P4 — "External CTA"
CTA parece vir "de fora" sem ser conclusão lógica do que foi lido.
- Fix: Adicionar "Summary of Updated Beliefs" antes do CTA: 3-4 pontos que resumem o que o ICP "descobriu" lendo a copy
