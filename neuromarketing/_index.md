## Mapa de lastro — o porquê científico de cada filtro F1–F7

Este acervo é a EVIDÊNCIA dos filtros do `../copy-brain/` (a execução com nota fica lá; aqui mora o porquê).

| Filtro | Threshold | Conceito-fonte | Arquivo |
|---|---|---|---|
| **F1 Atenção** | ≥0.75 | atenção é o gargalo (Simon) + dwell-time Meta + thumb-stopping 0,25–0,4s | `economia-da-atencao.md` |
| **F2 Emoção** | ≥0.79 | amígdala 50–100ms + paradoxo Pepsi/vmPFC + rostos 4x | `neurociencia-do-consumo.md` |
| **F3 Memória** | ≥0.72 | vmPFC + hipocampo (marca sobrepõe o sabor) | `neurociencia-do-consumo.md` |
| **F4 Decisão** | ≥0.75 | aversão à perda 2x, decoy, paradoxo da escolha | `sistema1-sistema2-decisao.md` |
| **F5 Linguagem** | ≥0.73 | carga cognitiva (Sweller) + sound-off/legenda | `economia-da-atencao.md` |
| **F6 Recompensa** | ≥0.75 | dopamina/VTA, erro de predição de recompensa | `neurociencia-do-consumo.md` |
| **F7 Predição** | ≥0.71 | predictive processing (Friston) + neuroscore + fricção cognitiva | `ia-preditiva-neuroscore.md` |
