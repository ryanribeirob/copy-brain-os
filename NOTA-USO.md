# NOTA DE USO — tribev2-main

**O que é**: Fork do [TRIBE v2](https://github.com/facebookresearch/tribev2) da Meta Research.
Modelo de deep learning para **predição de respostas cerebrais (fMRI)** a vídeos, áudio e texto.
É código científico de ML que requer PyTorch, pesos de modelo no HuggingFace e dados de cortex mesh.

**O que NÃO é**: Biblioteca de copy patterns, neuroscience aplicada ao marketing ou
frameworks de persuasão. O nome do modelo pode enganar quem procura conteúdo de copy.

**Status**: Código científico de ML pesado. Requer GPU e os pesos da Meta; fica fora do fluxo padrão de copy.

**Se você quer neuroscience aplicada ao copy/marketing**: use a camada Copy Brain OS deste repositório — os 7 filtros F1–F7 em `copy-brain/` e a base de evidência em `neuromarketing/`. É onde os princípios de neurociência viram decisão de copy, sem precisar rodar o modelo neural.
